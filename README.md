# Fish Audio Preprocessor

这个 Repo 包含了一些用于处理音频的脚本. 主要包含以下功能:

- [x] 视频/音频转 wav
- [x] 音频人声分离
- [x] 音频自动切片
- [x] 音频响度匹配
- [ ] 音频人声匹配
- [x] 音频数据统计
- [x] so-vits-svc 数据集处理

([ ] 表示未完成, [x] 表示已完成)

**本代码已在 Ubuntu 22.04 / 20.04 + Python 3.10 测试过, 如果在其他版本遇到问题, 欢迎反馈**

## 上手指南:

```
pip install -e . 
fap --help
```
